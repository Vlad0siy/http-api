Для работы приложения нужно запустить main.exe и в строке запроса браузера localhost:6060, в этом случае будет 404 not found. 
Обращаться к методам следует: localhost:6060/add?a=1&b=1 - сумма двух чисел a и b соответсвенно. 
Доступные команды: add - сложение, sub - вычитание, div - деление, mul - умножение.
